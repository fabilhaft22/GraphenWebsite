01 — Grundlagen

Was ist ein Graph?

Ein Graph G = (V, E) ist eine mathematische Struktur, die aus einer Menge von Knoten (Vertices, V) und einer Menge von Kanten (Edges, E) besteht. Kanten verbinden jeweils zwei Knoten miteinander und modellieren Beziehungen oder Verbindungen zwischen Objekten.

Graphen sind allgegenwärtig: Straßennetze, soziale Netzwerke, das Internet, Stammbäume oder Computernetzwerke lassen sich alle als Graphen darstellen.

Ungerichteter Graph

Kanten haben keine Richtung. Die Verbindung (A, B) gilt in beide Richtungen. Beispiel: Freundschaft in sozialen Netzwerken.

Gerichteter Graph (Digraph)

Kanten haben eine Richtung (Pfeil). A → B bedeutet nicht zwingend B → A. Beispiel: Weblinks, Einbahnstraßen.

Knoten = Objekt/Punkt
Kante = Verbindung
Grad = Anzahl der Kanten eines Knotens
Pfad = Folge von Knoten über Kanten
Zyklus = Pfad, der zum Start zurückführt
Gewicht = Kosten einer Kante

BFS

// Breitensuche · Breadth-First Search

Die Breitensuche erkundet den Graphen ebenenweise: Zuerst alle Nachbarn des Startknotens, dann deren Nachbarn usw.

BFS findet immer den kürzesten Pfad (nach Kantenanzahl) in ungewichteten Graphen.

📦 Datenstruktur: Queue (FIFO)
function BFS(graph, start): queue ← [start] visited ← {start} while queue not empty: node ← queue.dequeue() besuche(node) for neighbor in graph[node]: if neighbor not in visited: visited.add(neighbor) queue.enqueue(neighbor)

DFS

// Tiefensuche · Depth-First Search

Die Tiefensuche verfolgt jeden Pfad so weit wie möglich, bevor sie zurückgeht (Backtracking).

DFS eignet sich gut für Zykluserkennung, topologische Sortierung und Zusammenhangskomponenten.

📚 Datenstruktur: Stack (LIFO)
function DFS(graph, start): stack ← [start] visited ← {} while stack not empty: node ← stack.pop() if node not in visited: visited.add(node) besuche(node) for neighbor in graph[node]: if neighbor not in visited: stack.push(neighbor)

04 — Kürzeste Wege

Dijkstra-Algorithmus

Der Dijkstra-Algorithmus findet den kürzesten Weg von einem Startknoten zu allen anderen Knoten in einem gewichteten Graphen (mit nicht-negativen Kantengewichten).
